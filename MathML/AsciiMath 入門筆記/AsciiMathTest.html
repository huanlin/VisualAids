國一上 指數律
已知 m、 n 為正整數，利用乘方介紹指數符號 `a^n`，得到指數律：
(1) `a^m * a^n = a^(m+n)`
(2) `a^m -: a^n = a^(m-n)`
(3) `(a^m)^n = a^(mn)`
(4) `(ab)^n = a^n * b^n`
(5) `(a/b)^n = a^n/b^n`
1. `a=(-5/3)^2, b=(-5/3)^3, c=(-5/3)^4, d=(-5/3)^5`, 則 a、b、c、d 的大小順序為何？
(A) a > b > c > d
(b) a < b < c < d
(c) c > a > b > d
(d) c > a > d > b
